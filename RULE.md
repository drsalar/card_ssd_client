## 十三张规则

### 基础规则

1. 各玩家将十三张牌排成三道，分别称为：
   - **头道**：3 张牌
   - **中道**：5 张牌
   - **尾道**：5 张牌
2. 三道牌的大小必须满足：**头道 < 中道 ≤ 尾道**。
3. 各玩家排好牌后，打开牌并两两比较：
   - 头道与头道比
   - 中道与中道比
   - 尾道与尾道比
4. 比较时先比牌型；牌型相同时，再比点数。
   - 点数比较仅适用于：**同花、三条、对子**。

### 普通牌型

> 以下牌型从大到小排列，规则类似德州扑克。

| 牌型 | 说明 |
| --- | --- |
| 五龙 | 5 张相同的牌组成；在加色的情况下会出现。 |
| 同花顺 | 5 张相同花色且数字连续的牌。 |
| 炸弹 | 4 张相同点数的牌，配上 1 张杂牌。 |
| 葫芦 | 3 张相同点数的牌，配上一对。 |
| 同花 | 5 张花色相同的牌。 |
| 顺子 | 5 张数字连续的牌，A 可以当 1 使用。 1-5 的顺子仅比 10-A 的顺子小|
| 三条 | 3 张相同点数的牌，配上 2 张杂牌。 |
| 两对 | 2 组对子，配上 1 张杂牌。 |
| 对子 | 2 张同点数的牌，俗称“轮车”。若为头道，需外加任意 1 张单牌；若为中道或尾道，需外加任意 3 张牌。 |
| 散牌 | 即德州扑克中的“高牌”。所有牌均为单张，未组成上述任何牌型。比较时按点数从大到小依次比较。头道仅可能为散牌/对子/三条。 |

### 多组对子称呼

| 对子组数 | 称呼 |
| --- | --- |
| 3 组对子 | 三轮车 |
| 4 组对子 | 四轮车 |
| 5 组对子 | 五轮车 |

### 特殊加分

> 特殊加分由**输给此玩家者**支付。

| 情况 | 额外支付分数 |
| --- | ---: |
| 冲三（头道三条） | 2 分 |
| 中道葫芦 | 1 分 |
| 中道炸弹 | 3 分 |
| 中道同花顺 | 4 分 |
| 中道五龙 | 9 分 |
| 尾道炸弹 | 3 分 |
| 尾道同花顺 | 4 分 |
| 尾道五龙 | 9 分 |

### 打枪与本垒打

- **打枪**：若一名玩家三道皆胜于另一名玩家，则可打枪该玩家，整体积分加倍。
- **本垒打**：若一名玩家打枪了对局中**所有对手**（2 人场 = 打枪对方 1 人；3-6 人场 = 打枪所有其余玩家），则在打枪基础上再 ×2 加倍。

### 其他规则

- **马牌**：当一对参与比牌的玩家中**任意一方**持有马牌（红桃 5）时，该 pair 中**双方**在该对比牌中的所有得分（含特殊加分、打枪 ×2、本垒打 ×2）整体再 ×2，与打枪 / 本垒打的倍率累乘生效。换言之，马牌的加倍作用于"对局"而非"持有者个人"。
- **结算分项一致性**：本局结算面板中，每位玩家三道（头道 / 中道 / 尾道）的得分已合并本道特殊加分及全部倍率，三道相加 == 本局总分。

### 超过四人时的加色规则

超过四人时，可以根据人数增加花色。
例如：5 人游戏时，可以加一组方块的 `2~A`，6 人游戏时，可以各加一组方块和草花的 `2~A`。

加色后，以下牌型会出现特殊比较规则：

| 牌型 | 特殊比较规则 |
| --- | --- |
| 同花 | 带对的同花大于普通同花；带 2 个对的同花大于带 1 个对的同花。 |
